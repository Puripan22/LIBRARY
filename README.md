**## Getting Started**

1.  Install dependencies:
    
    Bash
    
    ```
    npm install
    
    ```
    
    โปรดใช้โค้ดด้วยความระมัดระวัง  [ดูข้อมูลเพิ่มเติม](https://bard.google.com/faq#coding)
    
    content_copy
    
2.  Start the development server:
    
    Bash
    
    ```
    npm run dev
    
    ```
    
    โปรดใช้โค้ดด้วยความระมัดระวัง  [ดูข้อมูลเพิ่มเติม](https://bard.google.com/faq#coding)
    
    content_copy
    
3.  Open  `http://localhost:5173`  in your browser.
